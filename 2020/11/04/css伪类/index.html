css伪类

2020.11.04 weizhi 前端  热度 ℃

最近给伪类坑了一下 写下来长个记性

:link

适用于未访问过的链接

:visited

用于已被访问过的链接

:hover

鼠标悬浮在连接上时触发

:active

鼠标点击链接激活时触发

:focus

目标成为焦点时触发

注意：a:hover 必须位于 a:link 和 a:visited 之后

状态伪类

  • :enabled 选择已启用元素
  • :disabled 选择被禁用元素
  • :checked 选择选中的输入元素（单选按钮和复选框）

这三个伪类主要是用于表单元素的操作

:first-child/:last-child

选择属于其父元素的第一个/最后一个子元素

:nth-child(n)

选择属于其父元素的第n个子元素

:nth-of-type(n)

选择属于其父元素的特定类型的第n个子元素的每个元素
说起来有点绕口，看例子理解吧


个人理解就是：在所有节点中找标签为p的，然后将找到的第二个p标签背景色变为黄色
